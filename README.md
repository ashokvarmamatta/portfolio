# Matta Ashok Varma — Portfolio

## How to View
Simply open `index.html` in any modern browser (Chrome, Firefox, Edge, Safari).

No build step, no server, no dependencies needed — it's fully self-contained.

## Files
- `index.html` — Complete portfolio website (self-contained)
- `profile.json` — Structured profile data (for future edits)
- `PLAN.md` — Generation plan and data sources
- `README.md` — This file

## Deploy to GitHub Pages
1. Create a new repository on GitHub
2. Upload `index.html` to the repository
3. Go to Settings > Pages > Source: Deploy from branch (main)
4. Your portfolio will be live at `https://yourusername.github.io/repo-name`

## Deploy to Vercel / Netlify
1. Drag and drop the folder to vercel.com/new or netlify.com/drop
2. Done — instant deployment with HTTPS
